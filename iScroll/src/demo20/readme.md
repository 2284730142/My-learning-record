本篇为分zoom缩放设置

补充：缩放 为了使用缩放功能，你最好使用iscroll-zoom.js脚本。

1.基本设置
    
        a.options.zoom
        此属性设置为true启用缩放功能。
        默认值：false
        
        b.options.zoomMax
        最大缩放级数。
        默认值：4
        
        c.options.zoomMin
        最小缩放级数。
        默认值：1

        d.options.startZoom
        初始缩放等级
        默认值：1
        
        e.options.wheelAction	
        滚轮动作
        设为’zoom’，可以用滚轮缩放
        undefined
